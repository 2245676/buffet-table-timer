# 安卓设备访问指南

本系统支持在安卓设备上访问和使用，界面已针对移动端进行了响应式优化。

## 方式一：局域网访问（推荐）

### 步骤：

1. **确保电脑和安卓设备在同一WiFi网络下**

2. **启动服务器**
   - 在电脑上运行启动脚本（Windows双击`start-windows.bat`，Linux/Mac运行`./start-linux.sh`）
   - 等待服务器启动成功

3. **查找电脑IP地址**
   
   **Windows系统：**
   ```cmd
   ipconfig
   ```
   找到"IPv4 地址"，例如：`192.168.1.100`

   **Linux/Mac系统：**
   ```bash
   ifconfig
   # 或
   ip addr
   ```
   找到类似 `192.168.1.100` 的地址

4. **在安卓设备上访问**
   - 打开浏览器（推荐Chrome或Firefox）
   - 输入地址：`http://电脑IP地址:3000`
   - 例如：`http://192.168.1.100:3000`

5. **添加到主屏幕（可选）**
   - 在浏览器菜单中选择"添加到主屏幕"
   - 这样可以像APP一样快速启动

## 方式二：使用Manus云端部署

如果您需要在任何地方访问系统（不限于局域网），可以使用Manus的云端部署功能：

1. 在Manus管理界面点击"Publish"按钮
2. 系统会生成一个公网访问地址（例如：`https://xxx.manus.space`）
3. 在任何设备的浏览器中访问该地址即可

## 移动端优化特性

本系统已针对移动端进行了以下优化：

- ✅ 响应式布局：自动适配不同屏幕尺寸
- ✅ 触摸优化：按钮和卡片大小适合手指点击
- ✅ 流畅动画：优化了移动端的动画性能
- ✅ 实时更新：支持移动端的实时数据刷新
- ✅ 离线提示：网络断开时会显示友好提示

## 常见问题

### Q: 无法访问服务器？
A: 请检查：
1. 电脑和手机是否在同一WiFi网络
2. 电脑防火墙是否允许3000端口访问
3. IP地址是否输入正确

### Q: 页面显示不正常？
A: 请尝试：
1. 刷新页面（下拉刷新）
2. 清除浏览器缓存
3. 使用推荐的浏览器

### Q: 如何在多个设备同时使用？
A: 系统支持多设备同时访问，所有设备会实时同步桌台状态。
